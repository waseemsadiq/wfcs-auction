Quick Start Prompts
Built-in Claude Prompts

These prompts are registered directly in the MCP server. Use /prompts in Claude to access them.

daily-briefing

Today's bid count, revenue, live status, and outstanding payments — in one go.

auction-report

Full report for a specific auction: items, bids, revenue, outstanding payments.

revenue-summary

Revenue overview including Gift Aid, formatted for charity trustees.
